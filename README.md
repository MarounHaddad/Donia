# Donia

An isometric city-building game set in ancient Egypt.

Assets and code for a game I worked on for a bit.

Last tested on Unity 2017.4.16.f1  

Code: C#  
Buildings: Sketchup 3D  
People: Fuse  
People Animation: Mixamo + Blender
